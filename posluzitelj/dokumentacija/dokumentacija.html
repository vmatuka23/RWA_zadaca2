Napomene i poznati problemi

Zadaća 1 - Poznati problemi (Web stranice)

  • Manje validacijske greške na nekim formama
  • Poboljšanja korisničkog iskustva moguća u budućim verzijama

Zadaća 2 - Trenutno stanje (Angular aplikacija)

  • Angular aplikacija sadrži samo osnovni template bez implementiranih funkcionalnosti Task 2
  • Sve funkcionalnosti Task 1 su implementirane samo na web stranicama (posluzitelj/build/aplikacija/html)
  • REST API servisi na poslužitelju su implementirani i funkcionalni
  • Nedostaje integracija Angular aplikacije s REST API-jima
  • Nedostaje implementacija email aktivacije
  • Nedostaje implementacija pretrage i filtriranja multimedijalnog sadržaja u Angular aplikaciji
  • Dokumentacija je dostupna na ruti /dokumentacija u Angular aplikaciji

Struktura projekta

  • Zadaća 1 - Web stranice (HTML/JS): posluzitelj/src/aplikacija/ i posluzitelj/build/aplikacija/
  • Zadaća 2 - Angular aplikacija: angular/projekti/zadaca2/
  • REST API poslužitelj: posluzitelj/src/servis/ - Svi endpointi implementirani i funkcionalni
